## 智能微秘书（微信每日说升级版3.0）

让你闲置的微信号成为你的智能小秘书（没有闲置的也没关系，添加我的小助手微信号，给你分配一个小秘书）。

帮你创建定时任务，每日提醒，纪念日提醒，当日提醒。当然基础的给女朋友的每日说功能也是必备的，群机器人聊天，垃圾分类，天气查询，土情话查询，老黄历查询，顺口溜查询等众多功能

## 功能及部署介绍

更多功能及配置请移步：[智能微秘书介绍](https://www.xkboke.com/web-inn/secretary/client.html)


## 配置

`env.js` 文件中填入智能微秘书的apiKey 和 apiSecret [注册地址](https://wechat.aibotk.com/#/signup)

## 运行

项目根目录

第一步 `npm install`

第二步 `npm run pm2`(如果报错pm2找不到，请执行`npm install pm2 -g` 后重新执行此步骤)
